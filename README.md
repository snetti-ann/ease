# EASA ATPL Quiz Helper

Простой веб-инструмент для обработки вопросов уровня EASA ATPL (Airline Transport Pilot Licence).

## Что делает

- Парсит текст вопроса + 4 варианта ответов из textarea
- Отправляет вопрос в LLM с контекстом, что это EASA ATPL теоретический экзамен
- Получает из AI правильный вариант (A/B/C/D) и вставляет его в таблицу
- Хранит данные в `localStorage`
- Позволяет редактировать строки, удалять и экспортировать таблицу в CSV (Excel)

## Установка

1. Установите зависимости:

```bash
npm install
```

2. Установите ключ в файл `.env` (рекомендуется) или экспортируйте переменную окружения.

- Вариант (файл `.env`, сохраняется в проекте):

```
GOOGLE_API_KEY="ваш_ключ"
```

- Вариант (экспорт переменной в терминале):

```bash
export GOOGLE_API_KEY="ваш_ключ"
```

3. Запустите сервер:

```bash
npm start
```

4. Откройте браузер: http://localhost:5173

## Как пользоваться

1. Вставьте вопрос и варианты ответа (по одной строке):

```
Вопрос
A) вариант
B) вариант
C) вариант
D) вариант
```

2. Нажмите **"Добавить в таблицу"**. Система автоматически отправит запрос к AI и заполнит столбец **Правильный ответ**.
3. Экспортируйте данные в Excel через **"Экспорт в Excel"**.

## Примечания

- Если один и тот же вопрос уже есть в таблице, он не будет продублирован.
- Если модель возвращает что-то нестандартное, приложение попытается выделить A/B/C/D или 1/2/3/4.
